# Michel's Project Dashboard

A production-ready project management dashboard inspired by Nim's Dashboard with smooth animations and pixel-perfect design.

## Tech Stack
- Frontend: Next.js 14 (App Router)
- Language: TypeScript (strict mode)
- Styling: Tailwind CSS (dark mode only)
- Drag & Drop: @dnd-kit/core + @dnd-kit/sortable
- State Management: Zustand + localStorage persistence
- Icons: lucide-react
- Date Handling: date-fns

- Responsive sidebar with project list and user profile
- Top navigation bar with search and filters

- Smooth transitions between views

## Design Guidelines
- **Color Palette:**
  - Background: slate-900, slate-800
  - Text: slate-100, slate-300
  - Accent: orange-500, orange-600
  - Success: green-500
  - Warning: yellow-500
  - Error: red-500
  - Borders: slate-700

- **Typography:**
  - Font: System fonts (default Next.js)
  - Headers: font-semibold
  - Body: font-normal
  - Sizes: text-xs, text-sm, text-base, text-lg, text-xl

- **Spacing:**
  - Consistent use of Tailwind spacing scale
  - Card padding: p-4
  - Column gap: gap-4
  - Section margins: mt-6, mb-6

- **Animations:**
  - Transitions: transition-all duration-200 ease-in-out
  - Hover: hover:scale-105, hover:shadow-lg
  - Drag: opacity-50 during drag
  - All animations must be 60fps smooth

  status: 'todo' | 'in-progress' | 'done';

  assignee: {
    name: string;
    avatar: string;
  };
  dueDate: Date;
  tags: string[];
  createdAt: Date;
  updatedAt: Date;
}
